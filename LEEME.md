Los valores pueden ser ingresados a la calculadora por medio de darle click al número para hacer esto se creó una función que se llama ingresar_valores.
También se pueden ingresar valores al presionar el numero o la operación directamente en el teclado para ello se creó una función que recibe el nombre de ingresar_valores_teclado.
También se crearon funciones para borrar un solo elemento de los ingresados o para borrarlos todos.
Se creo la función raiz_cuadrada para obtener las raíces cuadradas de los números ingresados.
La calculadora es adaptable, es decir, si se amplía la ventana de está los botones se van adaptar a la maximación.
